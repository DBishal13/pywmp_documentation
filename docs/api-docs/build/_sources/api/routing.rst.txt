Simulation Module
=================

The :mod:`pywmp.simulate` module provides hydrologic simulation functionality.

HydrologicSimulator Class
-------------------------

The HydrologicSimulator class orchestrates the complete hydrologic simulation workflow.

.. note::
   The complete API documentation will be available when the package is properly installed.
   
   Key classes in this module:
   
   - ``HydrologicSimulator`` - Main simulation orchestration class

Kinematic Wave Routing
----------------------

Flow routing capabilities include various methods for channel and overland flow simulation.

Time-Area Method
----------------

Functions for time-area routing methods:

- ``time_area_method()`` - Time-area routing calculations
- ``calculate_time_of_concentration()`` - Time of concentration calculations

Dynamic Wave Routing
---------------------

Advanced routing methods for complex flow situations.

Reservoir Routing
-----------------

Functions for reservoir and storage routing:

- ``reservoir_routing()`` - Reservoir flow routing
- ``level_pool_routing()`` - Level pool routing method

Channel Hydraulics
------------------

Functions for hydraulic computations:

- ``manning_velocity()`` - Calculate velocity using Manning's equation
- ``calculate_hydraulic_radius()`` - Hydraulic radius calculations
- ``normal_depth()`` - Normal depth calculations
