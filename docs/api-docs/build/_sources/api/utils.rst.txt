Structure Flows Module
======================

The :mod:`pywmp.structure_flows` module provides hydraulic structure flow calculations.

Pump Discharge
--------------

Functions for calculating pump discharge rates and hydraulic structure flows.

.. note::
   The complete API documentation will be available when the package is properly installed.
   
   Key functions in this module:
   
   - ``pump_discharge()`` - Calculate pump discharge rates
   - ``gated_spillway_discharge()`` - Calculate spillway discharge

File I/O Operations
-------------------

Functions for reading and writing geospatial data:

- ``write_geotiff()`` - Export raster data to GeoTIFF format
- ``export_shapefile()`` - Export vector data to Shapefile format
- ``read_rainfall_data()`` - Read rainfall time series data

Coordinate Transformations
--------------------------

Functions for coordinate system transformations:

- ``transform_coordinates()`` - Transform between coordinate systems
- ``utm_to_geographic()`` - Convert UTM to geographic coordinates
- ``calculate_utm_zone()`` - Determine appropriate UTM zone

Data Processing
---------------

Functions for DEM and raster processing:

- ``resample_dem()`` - Resample DEM to different resolution
- ``fill_sinks()`` - Fill sinks in elevation data
- ``smooth_surface()`` - Apply smoothing filters to surfaces
- ``clip_raster()`` - Clip raster data to study area

Validation and Quality Control
------------------------------

Functions for data validation and quality assurance:

- ``validate_dem()`` - Validate DEM data quality
- ``check_network_connectivity()`` - Check network topology
- ``quality_control_report()`` - Generate QC reports

Mathematical Operations
-----------------------

Functions for statistical and mathematical operations:

- ``interpolate_values()`` - Interpolate missing values
- ``calculate_statistics()`` - Calculate descriptive statistics
- ``fit_distribution()`` - Fit probability distributions
