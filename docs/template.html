Code Example

Here's how to use PyWMP in your projects:

import pywmp

# Create a watershed model
model = pywmp.WatershedModel()

# Load data
model.load_dem("watershed.tif")
model.load_rainfall("rainfall.csv")

# Run simulation
results = model.simulate()

# Export results
results.export("output.csv")
